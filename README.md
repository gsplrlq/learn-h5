## 在线预览 Preview

👓 [点击这里](https://yulimchen.github.io/vue3-h5-template/)（PC浏览器请切换手机端模式）

## 运行项目

```shell
# 克隆项目
git clone https://github.com/gsplrlq/learn-h5.git

# 进入项目目录
cd learn-h5

# 安装依赖
pnpm install

# 启动服务
pnpm dev
```
